# Data-Science-Projects
A collection of interactive data science dashboards developed using Python, Meteostat, Pandas, Altair, and Quarto. These projects explore weather trends, wind behavior, and regional climate patterns through data visualization, exploratory analysis, and machine learning techniques.

## Project Overview

This repository contains three weather-focused analytics dashboards built using real-world meteorological data. Together, these projects demonstrate skills in:

- Data Collection and Cleaning
- Exploratory Data Analysis
- Data Visualization
- Dashboard Development
- Statistical Analysis
- Machine Learning
- Geographic Data Analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Altair
- Quarto
- Meteostat API
- Scikit-Learn

# Dashboard 3: U.S. Weather Analytics Dashboard

A comprehensive weather analytics project investigating regional weather patterns, geographic influences, and extreme weather events across the United States.

### Features

- Multi-region weather comparisons
- Tornado case study analysis
- Geographic weather comparisons
- Interactive dashboards
- Machine learning classification models

### Machine Learning

Models were developed to classify weather conditions using meteorological measurements such as:

- Temperature
- Humidity
- Pressure
- Wind Speed
- Wind Direction
- Precipitation

Algorithms evaluated include:

- Logistic Regression
- Random Forest
- HistGradientBoosting

---

## Key Findings

- Weather patterns vary significantly across U.S. regions.
- Geographic features such as mountains, lakes, deserts, and coastlines influence local weather conditions.
- Midwest locations experienced some of the highest average wind speeds.
- Extreme weather events can significantly impact short-term weather behavior.
- Machine learning models can successfully classify weather conditions using historical weather observations.
